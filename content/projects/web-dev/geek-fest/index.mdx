## Overview

**GeekFest** is a modern event management platform that caters to tech and gaming enthusiasts. Users can discover events, bookmark favorites, manage their profiles, and explore a wide catalog of services — all with secure authentication via Firebase.

## Key Features

### Authentication

- Email/password login
- Google sign-in support
- Firebase-secured user sessions

### Event Bookmarking

- Save favorite events with a single click
- Access personalized list of bookmarked events

### Profile Management

- View and edit display name, profile photo, and account details
- Seamless user experience with Firebase data binding
