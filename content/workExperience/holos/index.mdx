##### [HOLOS-IE / HOLOS-EU](https://www.ucd.ie/holos-ie/)

Software Developer focused primarily on **backend and desktop application development** using .NET technologies, with additional contributions to modernization efforts involving cross-platform and AI-assisted tooling. Worked on domain-heavy **environmental modeling systems** combining agriculture, economics, and emissions accounting.

- Developed and maintained core application modules using **C#, Avalonia UI, and Telerik UI**, supporting precision agriculture and decision-support workflows.
- Implemented **agroforestry** and **dairy cattle economics** models, including cost–revenue analysis across multi-period farm management cycles.
- Designed and documented **IPCC Tier 2–based emission factor models**, including **soil organic carbon (SOC)** and **nitrogen pool dynamics** (initiation, decay, and steady-state behavior).
- Conducted **research and data engineering** on large-scale environmental and agricultural datasets, supporting model accuracy and system validation.
- Contributed to the **HOLOS-EU modernization initiative**, building a cross-platform demo using **Electron, React, TypeScript, TailwindCSS, FastAPI**, and AI-assisted analytics to improve usability and scalability.
